## Customization

### Business Information
Update the following in each HTML file:
- Phone number in `tel:` links
- Email address in `mailto:` links
- Business address in footer and structured data

### Brand Colors
Modify CSS variables in `css/style.css`:
```css
:root {
  --primary-blue: #00D3F2;
  --primary-green: #0A8100;
  --accent-orange: #E87600;
}
```
